# 基础模块实践项目

本目录包含基础模块的实践项目，用于巩固所学知识。

## 项目列表

1. **simple-js-bundle** - 简单的 JS 打包项目
2. **css-processing** - CSS 处理项目
3. **image-processing** - 图片资源处理项目

## 项目结构

每个项目包含：
- `template/` - 初始代码模板
- `solution/` - 参考答案
- `README.md` - 项目说明、目标和提示

## 使用方法

1. 阅读项目的 README.md 了解项目目标
2. 从 template 目录开始编写代码
3. 完成后可以参考 solution 目录的参考答案
