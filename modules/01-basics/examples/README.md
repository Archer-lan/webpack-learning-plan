# 基础模块代码示例

本目录包含基础模块各课程的可运行代码示例。

## 示例列表

- `01-installation-example/` - webpack 安装和初始化示例
- `02-first-project-example/` - 第一个 webpack 项目示例
- `03-entry-output-example/` - entry 和 output 配置示例
- `04-loaders-example/` - loader 使用示例
- `05-plugins-example/` - plugin 使用示例
- `06-modes-example/` - 开发模式和生产模式示例
- `07-dev-server-example/` - webpack-dev-server 使用示例
- `08-hmr-example/` - 模块热替换（HMR）示例

## 运行示例

每个示例目录都包含完整的项目文件和 README.md 说明文档。

运行步骤：
1. 进入示例目录
2. 安装依赖：`npm install`
3. 运行示例：按照各示例的 README.md 说明执行
