## 在线课程

### 英文课程
1. **Webpack Academy** (免费)
   - 网址：https://webpack.academy/
   - 内容：官方推荐的免费课程，涵盖基础到高级内容

2. **Udemy - Webpack 5: The Complete Guide**
   - 讲师：Stephen Grider

3. **Frontend Masters - Webpack Fundamentals**
   - 讲师：Sean Larkin (Webpack 核心团队成员)

## 学习路径建议

### 初学者（0-3 个月）
1. 阅读官方 Getting Started 文档
2. 观看 Traversy Media 的快速入门视频
3. 完成 Webpack Academy 的免费课程
4. 动手配置一个简单的项目

### 中级开发者（3-6 个月）
1. 阅读《深入浅出 Webpack》
2. 学习 Frontend Masters 的课程
3. 研究开源项目的 Webpack 配置
4. 编写自定义 loader 和 plugin

### 高级开发者（6 个月以上）
1. 阅读 Webpack 源码
2. 参与 Webpack 社区贡献
3. 优化大型项目的构建性能
4. 分享经验和最佳实践

## 持续学习

1. **关注官方更新**：定期查看官方博客和 Release Notes
2. **参与社区讨论**：在 GitHub Issues 和论坛中交流
3. **实践项目**：将学到的知识应用到实际项目中
4. **分享经验**：写博客或做技术分享
5. **阅读源码**：深入理解 Webpack 的实现原理

## 注意事项

- 优先选择最新版本的教程和文档
- Webpack 5 与 Webpack 4 有较大差异，注意版本区别
- 实践是最好的学习方式，多动手配置项目
- 遇到问题先查阅官方文档，再搜索社区解决方案
